# Wordbook Web 試作

このフォルダは、iPad ネイティブアプリ化に先立って UX・タイマー・難易度・筆跡 PNG 収集を検証するための Web プロトタイプです。

## 起動方法

同梱の `words.json` と同じフォルダで HTTP サーバーを立ち上げてブラウザで開いてください。

```powershell
# PowerShell 例
python -m http.server 8080
```

または VS Code の Live Server などを利用してください。

ブラウザで `http://localhost:8080` を開きます。

## 制限事項

- ブラウザ内では OCR は行いません。iPad 版では Apple Vision を使用します。
- ブラウザ用 OCR の結果は Apple Vision の品質根拠には使えません。
- 保護者 PIN、履歴、監査記録はブラウザの LocalStorage に保存されます。アプリ削除（サイトデータ消去）で失われます。

## 操作方法

- 縦画面を推奨します。
- Lv1 / Lv2 / Lv3 を選んでセッションを開始します。
- 制限時間はデフォルト 15 秒です（設定で変更可能）。
- 手書き後に「認識されない（手動正解）」を押すと、筆跡画像と問題情報が監査記録に保存され、正解扱いで次の問題へ進みます。
- 「書き直す」を押すと同じ問題を再出題します。
- 保護者画面では、初期 PIN `0000` で手動正解の記録を確認・確認済み/要注意の判定ができます。

## データ出力

セッション完了時または LocalStorage 保存データを JSON でダウンロードできます。筆跡画像は PNG 保存も可能です。
